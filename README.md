# Smart Shopping Mall Navigator

An intelligent web application that optimizes shopping routes using the Traveling Salesman Problem (TSP) algorithms.

## Features

- **Smart Route Optimization**: Multiple TSP algorithms (Greedy, Dynamic Programming, Genetic Algorithm)
- **Interactive Visualization**: Visual representation of the optimized route
- **Time vs Distance Optimization**: Choose to optimize for shortest distance or minimal time considering crowd levels
- **Smart Recommendations**: Get nearby offers and recommendations based on your shopping list
- **Modern UI**: Responsive design with Tailwind CSS

## Technologies Used

### Frontend
- React.js
- Tailwind CSS
- Axios
- Recharts
- React Select

### Backend
- Python Flask
- NumPy
- Flask-CORS
